# Changelog

All notable changes to this package will be documented in this file.

The format is based on [Keep a Changelog](http://keepachangelog.com/en/1.0.0/)
and this project adheres to [Semantic Versioning](http://semver.org/spec/v2.0.0.html).

## [1.6.1] - 2026-01-18

### Added

- Added github action to publish the wheel on pypi

## [1.6.0] - 2026-01-17

### Added

- Added complete setup to allow distribution of the package on PyPi
- Created helper script `jenkinsfilegenerator` to use `generator.main`

## [1.5.0] - 2026-01-14

### Added

- Allow features to have customization points to allow the users to output their own templates in the final output

## [1.4.1] - 2026-01-12

### Updated

- Made the blackboard data optional for the batch item

## [1.4.0] - 2026-01-08

### Added 

- Added a batch mode to generate multiple output files at once

## [1.3.0] - 2025-12-23

### Added 

- Added the blackboard data which allows to dynamically update the config files before generating the output

## [1.2.0] - 2025-10-20

### Added 

- Added plastic scm feature

## [1.1.0] - 2026-08-28

### Added 

- Added archive feature to store build artifacts on a shared folder or in an amazon S3 bucket

## [1.0.0] - 2025-07-29

### Added

- First version of the package.
